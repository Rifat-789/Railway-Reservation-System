# 🚆 Railway Reservation System (C Project)

A console-based Railway Reservation System developed in C. This application allows users to reserve seats, manage passenger information, and generate tickets efficiently using file handling and structured programming.

---

## 📌 Description

This project simulates a real-world railway booking system where users can:

- Book train seats
- Store and retrieve passenger data
- Generate tickets
- Manage reservations using file storage

It is designed for intermediate learners to strengthen core concepts like structures, file handling, and input validation in C.

---

##  Features

-  Seat Reservation System  
-  Seat Availability Check
-  Train Route Information
-  Passenger Record Management  
-  Search Passenger Information  
-  Ticket Generation  
-  Admin Pannel
-  File Handling (persistent storage using `.dat` files)  
-  Input Validation for safe operations  

---

## 🛠️ Technologies Used

- C Programming Language  
- Standard Libraries (`stdio.h`, `stdlib.h`, etc.)  
- File Handling (`fread`, `fwrite`)
